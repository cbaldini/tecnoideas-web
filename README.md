# Tecnoideas Web

Sitio web profesional de **Tecnoideas** - Consultora IT + Laboratorio de Ideas.

## 🚀 Características

- **Diseño moderno y responsivo** - Funciona en desktop, tablet y móvil
- **Estética clara y tecnológica** - Colores azul y cyan con fondo claro
- **Animaciones fluidas** - Logo flotante, parallax y transiciones suaves
- **Menú hamburguesa adaptable** - Optimizado para móviles
- **Secciones completas**:
  - Hero con propuesta de valor
  - Nosotros (Filosofía)
  - Qué hacemos (Servicios)
  - Productos (Robots, wearables, prototipos)
  - Experiencias (Casos: Alterna, Golosinas Aries)
  - Contacto

## 📁 Estructura

```
src/
├── index.html        # Contenido HTML
├── css/
│   └── style.css    # Estilos (colores, layout, animaciones)
├── js/
│   └── app.js       # Interactividad
└── img/             # Imágenes
```

## 💻 Uso

### Rápido (sin build)
Simplemente abrí `src/index.html` en el navegador.

### Con Node (npm)
```bash
npm install
npm run dev    # Desarrollo
npm run build  # Producción
```

## 🎨 Personalización

**Colores** en `src/css/style.css`:
- Primario: `#0066ff` (azul)
- Acento: `#00d4ff` (cyan)
- Fondo: `#fffafb` (blanco)

**Logo**: Reemplaza `teknoideas.png` y `TI.png`

**Contacto**: Actualiza email y teléfono en sección "Contacto"

## 📱 Responsive

- Desktop: 1200px+
- Tablet: 768px - 1199px
- Móvil: <768px
- Teléfono: <480px

## ♿ Accesibilidad

✓ HTML5 semántico
✓ ARIA labels
✓ Skip link para teclado
✓ Respeta `prefers-reduced-motion`
